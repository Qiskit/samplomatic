Added local C1 twirling support for `rzz` instructions for non-zero Clifford angles.
Added support for twirling parametric and non-Clifford `rzz` instructions with the new `parametric_zz` group mode. When an `rzz` is encountered, this option samples from the Pauli subgroup that commutes with `rzz`. For other entanglers, it defaults back to the Pauli group. This mode is set by annotating a box with a `Twirl` that has `Twirl.group_mode = parametric_zz` or by using a boxing pass manager with `twirling_group = parametric_zz`.
Added the `phase` group mode that samples uniformly from Paulis I and Z.
